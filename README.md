# 🏆 SportZie – Responsive Sports Landing Page

A modern, visually appealing, and fully responsive sports training landing page built using **HTML5, CSS3, and Vanilla JavaScript**.

This project was developed as **Task 01 – Responsive Landing Page** for the **Prodigy Infotech Web Development Internship**.

---

## 📌 Overview

SportZie is a fictional sports training platform designed to provide athletes with access to professional coaching, structured training programmes, and a supportive sports community.

The website features a premium modern design with smooth navigation, responsive layouts, interactive elements, and an engaging user experience across all devices.

---

## ✨ Features

- 📱 Fully responsive design for mobile, tablet, and desktop screens
- 🎨 Modern sports-themed UI with custom color palette and typography
- 🔝 Sticky navigation bar with scroll effects
- 📌 Active navigation link highlighting based on the current section
- 🍔 Responsive hamburger menu for smaller devices
- 🏃 Sports categories showcase
- 🏆 Training programmes section
- 👨‍🏫 Professional coaches showcase
- ⭐ Member testimonials section
- 🎯 Call-to-action section
- 🌊 Smooth scrolling navigation
- ✨ Interactive hover animations and transitions
